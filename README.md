### [Day 01](Day-01.py)
- **Introduction**: Basic statements, comments, and printing.
- **Variables**: Declaration, naming conventions (Camel, Pascal, Snake case), and multiple assignment.
- **Data Types**: `str`, `int`, `float`, `complex`, `list`, `tuple`, `range`, `dict`, `set`, `frozenset`, `bool`, `None`.
- **Type Casting**: Converting between different data types.

### [Day 02](Day-02.py)
- **Strings**: Slicing, modification (upper, lower, strip, replace, split), concatenation, and format strings (f-strings).
- **Escape Characters**: Using special characters in strings.
- **String Methods**: `capitalize`, `center`, `count`, `find`, `endswith`, `isupper`, etc.
- **Booleans**: evaluating values and expressions.

### [Day 03](Day-03.py)
- **Operators**: Arithmetic, Assignment, Comparison, Logical, Identity (`is`), Membership (`in`), Bitwise.
- **Operator Precedence**: Order of operations.
- **Lists**: Basics, accessing items, changing values, adding items (`append`, `insert`, `extend`), removing items (`remove`, `pop`, `del`, `clear`), and looping.
- **List Comprehension**: Concise syntax for creating lists.

### [Day 04](Day-04.py)
- **List Methods**: Sorting (`sort`, `reverse`, key functions), copying, and joining lists.
- **Tuples**: Creating, accessing, unpacking, looping, and methods (`count`, `index`).
- **Sets**: Creating, adding/removing items, operations (Union, Intersection, Difference, Symmetric Difference).
- **Dictionaries**: Accessing, changing, adding, removing items, looping, and nested dictionaries.
- **Control Flow**: `if`, `elif`, `else`, short-hand if-else.

### [Day 05](Day-05.py)
- **Match Case**: Python's switch-case equivalent.
- **Loops**: `while` loops, `for` loops, `break`, `continue`, `else` in loops, and nested loops.
- **Functions**: Defining functions, arguments, parameters, default values, keyword arguments, and return values.

### [Day 06](Day-06.py)
- **Advanced Arguments**: Arbitrary arguments (`*args`) and arbitrary accessible keyword arguments (`**kwargs`).
- **Scope**: Local vs Global scope, `global` and `nonlocal` keywords.
- **Decorators**: Function decorators, higher-order functions, and decorators with arguments.

### [Day 07](Day-07.py)
- **Lambda Functions**: Anonymous functions and their usage.
- **Functional Tools**: `map()`, `filter()`, and `sorted()` with lambda.
- **Recursion**: Recursive functions (Factorial, Fibonacci).
- **Generators**: `yield` keyword, creating generator functions for memory efficiency.

### [Day 08](Day-08.py)
- **Iterators**: Creating custom iterator classes with `__iter__` and `__next__`.
- **Modules**: Importing custom and built-in modules (`platform`), using `dir()`.
- **Datetime**: Working with dates and times.

### [Day 09](Day-09.py)
- **JSON**: Semantic syntax for data, parsing JSON (`loads`), and formatting Python objects to JSON (`dumps`).
- **Regex (Intro)**: Introduction to `re` module and simple search.

### [Day 10](Day-10.py)
- **Regular Expressions**: Detailed usage of `re` module.
- **Pattern Matching**: Character classes, special sequences, and quantifiers (`*`, `+`, `?`, `{n,m}`).

### [Day 11](Day-11.py)
- **Exception Handling**: `try`, `except`, `else`, `finally` blocks, and raising exceptions.
- **String Formatting**: Advanced f-strings (expressions, formatting types) and `.format()` method.
- **User Input**: Handling user input via `input()`.

### [Day 12](Day-12.py)
- **Virtual Environments**: Concepts and commands for setup.
- **OOP Basics**: Introduction to Classes and Objects.
- **Constructors**: The `__init__()` method and `self` parameter.

### [Day 13](Day-13.py)
- **Class Properties**: Modifying and deleting object properties.
- **Class vs Instance**: Class variables vs Instance variables.
- **Methods**: Defining methods, methods accessing and modifying attributes.
- **String Representation**: The `__str__()` method.

### [Day 14](Day-14.py)
- **Inheritance**: Parent and Child classes, `super()` function.
- **Polymorphism**: Method overriding and polymorphic behavior in functions and classes.
- **Encapsulation**: Private members (`__variable`) and using Getter/Setter methods.

### [Day 15](Day-15.py)
- **Inner Classes**: Defining classes within classes.
- **File Handling**: Reading files, writing/appending to files, creating and deleting files/directories using `open()` and `os` module.
